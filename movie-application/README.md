🎬 Movie Application

A modern React Native Movie App built with Expo and TypeScript.
Browse movies, explore details, and enjoy a smooth cross-platform experience on Android, iOS, and Web.

🚀 Features
🎥 Browse popular and trending movies
🔍 Search movies easily
❤️ Save favorite movies locally
📱 Cross-platform support (Android, iOS, Web)
⚡ Fast and responsive UI
🎨 Modern design with smooth navigation
🔐 Local storage using AsyncStorage
🌐 API integration with Axios
